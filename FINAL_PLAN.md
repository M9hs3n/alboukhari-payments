# الخطة النهائية المقترحة للتنفيذ

> النظام: تطبيق ويب احترافي يستبدل Google Sheets + Apps Script لإدارة دفعات الطلاب وإرسال التذكيرات.
> التقنية: **Laravel 12** + **MySQL** + **Redis** + **Livewire 3** + **Tailwind RTL**.
> المزوّد: **BulkGate** (موجود) — والواتساب يُضاف في المرحلة الثانية.

---

## ◾ الفكرة العامة (الرؤية الجديدة)

> **واجهة واحدة شبيهة بـ Google Sheets** — كل الطلاب وكل الأشهر أمام عينيك في صفحة واحدة. تضغط على أي خلية شهر فتفتح نافذة منبثقة سريعة لإدخال الدفعة بضغطتين. تضغط على اسم الطالب فتفتح لوحة جانبية بتفاصيله. الكل من نفس الشاشة.

### المبادئ الأساسية

1. **شاشة رئيسية واحدة** هي قلب النظام — جدول شامل بكل الطلاب وأشهرهم.
2. **النوافذ المنبثقة (Modals)** هي وسيلة الإدخال السريع — لا تنقل المستخدم لصفحة أخرى.
3. **اللوحات الجانبية (Side Panels)** لعرض التفاصيل دون مغادرة الشاشة الرئيسية.
4. **الصفحات المنفصلة** فقط للأشياء التي لا تنتمي للجدول: التقارير، القوالب، الإعدادات، سجل الحملات.
5. **عدّاد الرسائل ذكي** يظهر دائمًا قبل كل إرسال.

## أ) الجدول الرئيسي — مواصفاته

### الأعمدة
| العمود | الوصف |
|---|---|
| **id** | معرّف الطالب |
| **الاسم** | اسم الطالب + أيقونة إعدادات ⚙️ بجانبه |
| **الهاتف** | الرقم الأساسي (مختصر للعرض) |
| **👨‍👧 الإخوة** | شارة خضراء تظهر عدد الإخوة (إن وُجدوا) — قابلة للضغط |
| **يناير → ديسمبر** | ١٢ عمودًا، خلية لكل شهر |
| **رصيد** | المبلغ المتبقّي الإجمالي على الطالب |
| **حالة** | أيقونة (مخفي/محظور/مكاني/معلَّق) — تظهر فقط للحالات الخاصة |

### خلايا الأشهر — العرض والألوان
- 🟢 **30💵** — مدفوع نقدًا
- 🟢 **30🏦** — مدفوع بنكيًا
- 🟡 **15💵** — دفع جزئي نقدًا (15 من 30)
- 🔵 **0🏦** — سُجّل بـ ٠ بنكيًا (مستورد من الشيت)
- 🔴 **(فارغ)** — لم يدفع
- ⚫ **X** — متأخر (مرّ منتصف الشهر)
- ⬜ **(فارغ رمادي)** — شهر مستقبلي لم يحن بعد

### التفاعل مع الجدول
- ✅ **تجميد** الصفوف العليا والأعمدة اليمنى (الاسم/الهاتف) عند التمرير.
- ✅ **فرز** بالضغط على عنوان أي عمود.
- ✅ **فلاتر** سريعة على كل عمود.
- ✅ **بحث فوري** يبرز النتائج.
- ✅ **إخفاء/إظهار** أعمدة (مع حفظ تفضيلاتك).
- ✅ **عرض الكل** أو **١٠٠ صف لكل صفحة** أو **٥٠**.
- ✅ **تصدير** الجدول الحالي (مع الفلاتر المطبَّقة) إلى Excel.

## ب) الإدخال السريع للدفعات — النافذة المنبثقة (الأهم)

### كيف يعمل؟

**ضغطة واحدة على أي خلية شهر** → تفتح نافذة منبثقة صغيرة بسرعة فوق الخلية:

```
┌─────────────────────────────────────────┐
│  💶 دفعة جديدة                          │
│  أحمد الطيبي — يناير ٢٠٢٦              │
│  المستحق: 30€                           │
├─────────────────────────────────────────┤
│                                         │
│  المبلغ:  [   30   ]€                  │
│                                         │
│  الطريقة: ( 💵 نقد )( 🏦 بنكي )         │
│                                         │
│  ملاحظة: [_______________________]     │
│                                         │
│  التاريخ: [2026-05-19]                 │
│                                         │
│  [حفظ + التالي ⏎] [حفظ] [إلغاء]         │
└─────────────────────────────────────────┘
```

### مفاتيح اختصار سحرية
- **Enter** → حفظ والذهاب للخلية التالية تلقائيًا.
- **Tab** → الانتقال للحقل التالي داخل النافذة.
- **Esc** → إلغاء.
- **N** → اختيار نقد بضغطة.
- **B** → اختيار بنكي بضغطة.

### ميزات النافذة المنبثقة
- ✅ المبلغ يُملأ تلقائيًا بقيمة الرسم المستحق (يمكن تغييره).
- ✅ زر "حفظ + التالي" يحفظ ويُغلق ويفتح خلية الشهر التالي للطالب التالي (إدخال متسلسل سريع).
- ✅ إذا كانت الخلية تحتوي دفعة موجودة، النافذة تظهر للتعديل أو الحذف.
- ✅ سهم صغير لاستعراض الدفعات المتعددة في نفس الشهر (إن وُجدت).
- ✅ زر سريع "تكرار آخر دفعة" بنفس المبلغ وطريقة الدفع.

### وضع الإدخال السريع المتسلسل
زر "⚡ إدخال دفعات سريع" في الأعلى يُفعّل وضعًا خاصًا:
- بحث بالاسم في الأعلى.
- بعد كل دفعة، يعود المؤشر للبحث لاسم جديد.
- عدّاد: "✅ تم تسجيل ١٢ دفعة بمجموع 360€ خلال جلستك".
- يُفيد جدًا عند إدخال كثير من الدفعات في يوم تحصيل.

## د) اللوحة الجانبية — تفاصيل الطالب (بدون مغادرة الشاشة)

عند الضغط على اسم الطالب أو "عرض التفاصيل"، **لوحة جانبية** تنزلق من اليمين:

```
┌───────────────────────────────────────┐
│  أحمد الطيبي ✕                        │
│  👨‍👩‍👧‍👦 الإخوة: محمد، فاطمة (انقر للعرض)│
├───────────────────────────────────────┤
│  📞 +316XXXXXXX  / +316XXXXXXX        │
│  📧 (لا يوجد)                         │
│                                       │
│  💶 الرصيد: 60€                        │
│  📅 آخر دفعة: 2026-04-15 (30€ بنكي)   │
│  📨 آخر رسالة: 2026-05-01 (وصلت)       │
├───────────────────────────────────────┤
│  Tabs:                                │
│  [الدفعات] [الرسائل] [الإعدادات] [ملاحظات]│
│                                       │
│  ── الدفعات ──                        │
│  Jan: 30€ 💵 (2026-01-05)             │
│  Feb: 30€ 🏦 (2026-02-10)             │
│  Mar: 0€ 🏦 (سُجّل بنكيًا)              │
│  ...                                  │
├───────────────────────────────────────┤
│  [📲 أرسل]  [💶 ادفع]  [✏️ عدّل]      │
└───────────────────────────────────────┘
```

### Tabs داخل اللوحة الجانبية
1. **الدفعات** — كل أشهر السنة + كل التواريخ والطرق.
2. **الرسائل** — سجل كامل لكل رسالة سبقت للطالب.
3. **الإعدادات** — أزرار إخفاء/منع/مكاني/تعليق + رسم خاص للطالب.
4. **الملاحظات** — ملاحظات حرة عن الطالب (مفيد للمتابعة).

## هـ) شريط الإخوة — في اللوحة الجانبية وفي الجدول

### في عمود "الإخوة" في الجدول
- إذا كان للطالب إخوة: شارة 🟢 خضراء برقم عدد الإخوة.
- تمرير الماوس فوقها يعرض أسماءهم.
- الضغط يفتح صفحة العائلة في لوحة جانبية ثانية.

### في اللوحة الجانبية للطالب
- شريط أعلى اللوحة يعرض أسماء الإخوة كأزرار قابلة للضغط.
- بجانب كل اسم: لون حالته الحالية (أخضر/أصفر/أحمر).
- ضغطة على اسم أخ → تتبدّل اللوحة الجانبية لعرض الأخ.

### صفحة العائلة (لوحة جانبية)
- اسم ولي الأمر + الرقم + اللغة المفضّلة.
- جدول صغير بكل الأبناء وحالة شهرهم الحالي.
- زر "📲 أرسل للعائلة" (رسالة واحدة).
- زر "💶 دفعة عائلية" (مبلغ واحد يوزَّع تلقائيًا).

## و) شريط الأزرار العلوي — الإجراءات الجماعية

أزرار كبيرة واضحة في أعلى الشاشة الرئيسية:

### مجموعة الإرسال
1. **📨 إرسال جماعي** — لكل الطلاب المسموح لهم، نص حر.
2. **❓ إرسال لمن لم يدفع** — يفتح نافذة تختار فيها الشهر، ويعرض من سيستلم.
3. **⏰ إرسال للمتأخرين** — لمن مرّ منتصف الشهر ولم يدفع.
4. **💰 إرسال لمن دفع أقل من...** ← طلبك — تحدد المبلغ X.
5. **📊 إرسال لمن متبقي عليه أكبر من...** — رصيد متراكم > X.
6. **✋ إرسال لمحددين** — حدّد طلابًا بصناديق اختيار في الجدول ثم اضغط.

### مجموعة التحكم
7. **🔴 إيقاف فوري** — يوقف كل الإرسال الجاري.
8. **🟢 استئناف** — يكمل الإرسال من حيث وقف.
9. **🔄 كرّر آخر حملة** — بضغطة، نفس النص ونفس الشروط.

### مجموعة الإدارة
10. **⚡ إدخال دفعات سريع** — وضع الإدخال المتسلسل.
11. **➕ طالب جديد** — نافذة منبثقة لإضافة طالب.
12. **📥 استيراد** — رفع ملف Excel.
13. **📤 تصدير** — تنزيل الجدول الحالي.

## ز) شريط البحث والفلاتر

### البحث
- مربع بحث في الأعلى — يبحث بالاسم، الرقم، آخر ٤ أرقام، أو أي حقل ظاهر.
- البحث الفوري (Live) — لا تحتاج Enter.

### الفلاتر السريعة (قابلة للجمع)
- **الحالة هذا الشهر**: الكل / لم يدفع / دفع جزئي / مدفوع / متأخر.
- **سماح الرسائل**: الكل / مسموح / محظور / مكاني / معلَّق.
- **الإخوة**: الكل / مع إخوة / فرديين.
- **الرصيد**: الكل / متراكم > 30 / > 60 / > 90.
- **آخر دفعة**: قبل شهر / قبل ٣ أشهر / لم يدفع أبدًا.

### الفلاتر المحفوظة
- يمكنك حفظ تركيبة فلاتر باسم (مثل "المتأخرون أكثر من ٦٠€") والوصول لها بضغطة لاحقًا.

# 📑 الصفحات المنفصلة (المحدودة)

عدا الشاشة الرئيسية، لدينا فقط هذه الصفحات:

## ١) التقارير `/reports`
- ملخّص اليوم (KPIs).
- المتأخرون الآن.
- أكثر المتأخرين.
- التحصيل الشهري.
- توزيع الدفع (نقد/بنكي).
- تقرير الرسائل والتكلفة.
- تصدير أي تقرير Excel/PDF.

## ٢) القوالب `/templates`
- قائمة القوالب الجاهزة + إنشاء/تعديل.
- معاينة على طالب-عيّنة.
- عدّاد الحروف والرسائل.

## ٣) سجل الحملات `/campaigns`
- كل الحملات السابقة + حالتها + تكلفتها.
- ضغطة على حملة → لوحة جانبية بتفاصيلها (المستلمون، الفاشلون، النص).

## ٤) سجل الرسائل `/messages`
- كل الرسائل المرسَلة (مثل SMS_Log الحالي).
- فلاتر: التاريخ، الطالب، الحالة، النوع.
- بحث في النصوص.

## ٥) الإعدادات `/settings`
كل الإعدادات في صفحة واحدة بـ Tabs:
- عام (لغة، منطقة زمنية، عملة).
- الرسوم (الافتراضي، بداية السنة الدراسية).
- الإرسال (Sender ID، فرض ASCII، حد الساعة، سعر الرسالة).
- التذكير الدوري (تشغيل/إيقاف، ساعات).
- BulkGate (مفاتيح API مشفّرة).
- المستخدمون (لاحقًا إذا أردنا أكثر من مستخدم).

## ٦) سجل التغييرات `/audit`
- من غيّر ماذا ومتى (مفيد للمتابعة).

# ⚙️ الميزات الكاملة (مختصرة)

## ١) إدارة الطلاب
- استيراد من Excel بنفس بنية الشيت الحالي.
- تطبيع تلقائي لأرقام الهواتف الهولندية (E.164).
- أزرار التحكم لكل طالب: إخفاء، منع، مكاني، تعليق، استثناء.
- ملاحظات حرة، تاريخ التسجيل، تاريخ الانسحاب.
- بحث ذكي (يدعم الأخطاء الإملائية وآخر ٤ أرقام).

## ٢) نظام العائلة
- كشف تلقائي للإخوة (نفس رقم الهاتف).
- شريط الإخوة في اللوحة الجانبية.
- صفحة العائلة (لوحة جانبية ثانية).
- إرسال رسالة واحدة لكل عائلة.
- دفعة عائلية تُوزَّع تلقائيًا.
- متغيّرات قوالب للعائلة.

## ٣) الرسوم
- رسم افتراضي عام (مثلاً 30€).
- رسم خاص لطالب يطغى على العام.
- رسم خاص لطالب × شهر يطغى على الاثنين.
- رسوم إضافية (كتاب، رحلة) لطالب لشهر.

## ٤) الدفعات — مبسّطة
- إدخال من خلية الشهر مباشرة (نافذة منبثقة).
- زرّان فقط: 💵 نقد أو 🏦 بنكي.
- ملاحظة اختيارية.
- تاريخ قابل للتعديل.
- "حفظ + التالي" للإدخال المتسلسل السريع.

## ٥) الإرسال
- ٧ أنواع حملات (انظر §و).
- إرسال فردي من قائمة الطالب.
- إرسال للعائلة كلها برسالة واحدة.
- معاينة + Dry Run قبل أي إرسال.
- اختبار على رقمي قبل الإطلاق.

## ٦) عدّاد الرسائل
- مباشر أثناء الكتابة.
- ملخّص قبل الإرسال (عدد رسائل × عدد طلاب = الإجمالي).
- تقدير تكلفة بناءً على سعر الرسالة في الإعدادات.
- تنبيه أحمر عند تجاوز الحدّ.

## ٧) التذكير الدوري
- مفتاح عام + مفتاح لكل تذكير.
- اختيار القالب.
- ساعة التشغيل القابلة للتعديل.
- تشغيل يدوي عند الطلب.

## ٨) القوالب
- قوالب جاهزة بالهولندي (مطابقة للسكربت الحالي).
- متغيّرات: `{{Naam}}`, `{{month}}`, `{{المستحق}}`, `{{المدفوع}}`, `{{المتبقي}}`.
- متغيّرات العائلة: `{{أسماء_الأبناء}}`, `{{المبلغ_العائلي}}`.
- معاينة بأرقام حقيقية.
- عدّاد لكل قالب.

## ٩) التقارير والسجلات
- ملخّص اليوم + المتأخرون + شهري + توزيع طرق الدفع + تكلفة الرسائل.
- سجل كل الرسائل + سجل كل الحملات + سجل التغييرات.

## ١٠) الإعدادات
- كل شيء قابل للتعديل من الواجهة بدون لمس الكود.

## ١١) الأمان
- تسجيل دخول.
- HTTPS.
- نسخة احتياطية يومية.
- مفاتيح BulkGate مشفّرة.
